# Trivia Game

A simple trivia game complete with:

* Countdown Timer
* Animations 
* Question Images
* Final Score Readout

 Currently setup to test your knowledge of the Solar System. 

## Built With

* [JQuery](https://jquery.com/) - Event Handling & DOM manipulation
* [Bootstrap](https://getbootstrap.com/) - Front-end styling
